# File-Disguiser (Only Chinese Version)

---

## 中文文档 | Chinese Documentation

### 项目介绍
一款能够将文件伪装为图片，同时支持将伪装后的图片还原为原始文件的工具。

开发背景：在某些场景下，需要隐蔽地传输或存储文件，常规方式可能会受到限制或引起注意。通过将文件伪装成图片，可利用图片的普遍性进行合规传输，避免不必要的拦截或审查。本工具应运而生，旨在提供简单高效的文件伪装与还原解决方案。

### 核心功能
- 将任意类型的文件（文档、压缩包、程序等）伪装为常见图片格式（如PNG、JPG）
- 支持将伪装后的图片无损还原为原始文件，保证文件完整性
- 操作流程简洁，无需专业技术知识即可快速上手
- 伪装后的图片可正常预览，不影响常规图片查看体验

### 使用说明
1. **使用前准备**：确保待伪装的文件和作为载体的图片（可选）已准备就绪，建议提前备份重要文件。
2. **启动工具**：运行项目Disguiser/bin/Release目录下的可执行文件（或对应脚本），进入操作界面。

### 注意事项
- 伪装后的图片体积约等于原始图片体积与被伪装文件体积之和，过大文件可能导致图片体积异常。
- 还原文件时需使用本工具，使用其他工具可能导致还原失败或文件损坏。
- 请勿用于非法用途，使用本工具需遵守相关法律法规及平台规定。
- 对于部分特殊格式图片（如带有复杂元数据的图片），伪装后可能无法正常预览，属正常现象。

### 免责声明
本工具仅提供文件与图片的格式转换功能，不涉及数据加密或安全防护。使用过程中若因违规使用、操作不当导致的任何损失或法律责任，开发者不承担相关责任，请谨慎使用。

## English Documentation

### Project Description
A tool that can disguise files as images and support restoring the disguised images back to the original files.

Development Background: In some scenarios, there is a need to covertly transfer or store files, and conventional methods may be restricted or attract attention. By disguising files as images, they can be transmitted compliantly using the universality of images, avoiding unnecessary interception or review. This tool is developed to provide a simple and efficient solution for file disguise and restoration.

### Core Features
- Disguise any type of file (documents, compressed packages, programs, etc.) as common image formats (e.g., PNG, JPG)
- Support lossless restoration of disguised images to original files, ensuring file integrity
- Simple operation process, no professional technical knowledge required for quick start
- Disguised images can be previewed normally without affecting the regular image viewing experience

### Usage Instructions
1. **Preparation Before Use**: Ensure the file to be disguised and the image as a carrier (optional) are ready. It is recommended to back up important files in advance.
2. **Start the Tool**: Run the executable file in the release directory (Disguiser/bin/Release) to enter the operation interface.

### Notes
- The size of the disguised image is approximately equal to the sum of the size of the original image and the disguised file. Excessively large files may cause abnormal image size.
- This tool must be used for file restoration; using other tools may result in restoration failure or file damage.
- Do not use for illegal purposes, and comply with relevant laws, regulations and platform rules when using this tool.
- For some special format images (such as those with complex metadata), the disguised image may not be previewed normally, which is a normal phenomenon.

### Disclaimer
This tool only provides format conversion functions between files and images, and does not involve data encryption or security protection. The developer shall not be liable for any losses or legal liabilities caused by illegal use or improper operation during use. Please use it with caution.
